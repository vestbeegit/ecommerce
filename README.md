# bizora-ecommerce

A modern e-commerce platform built for businesses of all sizes.

## Features

- Full-featured e-commerce solution
- Modern and responsive design
- Easy to customize and extend
- Built on Laravel framework

## Getting Started

### Requirements

- PHP >= 8.1
- Composer
- MySQL/PostgreSQL
- Node.js and NPM

### Installation

1. Clone the repository
```bash
git clone https://github.com/muddassirmanzoor/bizora-ecommerce.git
cd bizora-ecommerce
```

2. Install dependencies
```bash
composer install
npm install
```

3. Configure environment
```bash
cp .env.example .env
php artisan key:generate
```

4. Run migrations
```bash
php artisan migrate
```

5. Start the development server
```bash
php artisan serve
```
